## 🚀 Quick Start

### Instalação
```bash
npm install
```

### Desenvolvimento
```bash
npm run dev
```
Abra: **http://localhost:5173**

### Build
```bash
npm run build
```

## 📁 Estrutura

```
src/
├── App.jsx                    # Router principal
├── components/Header.jsx      # Navegação
├── pages/
│   ├── Home.jsx              # /
│   ├── About.jsx             # /sobre
│   ├── Contact.jsx           # /contato
│   └── Prototype.jsx         # /prototipo
└── css/                      # Estilos
```

## 🎯 Rotas

| Rota | Página |
|------|--------|
| `/` | Home |
| `/sobre` | Sobre Nós |
| `/contato` | Contato |
| `/prototipo` | Protótipo |

## 🔧 Stack

- **React 19.2.0** - UI Library
- **Vite 7.3.1** - Build tool
- **React Router v6** - SPA routing
- **FontAwesome 6** - Ícones
- **CSS Puro** - Estilização
